# TASK: Visual Polish - The "High Voltage" Boost Effect

**Document Type:** UI/Visual Refinement
**Priority:** High
**Context:** The Ad Boost logic works, but the visual feedback is flat. We need to implement the "Power Grid" aesthetic to reward the user for boosting.
**Constraint:** **DO NOT** change the logic flow of `simulateAdWatch`. Only modify the Visuals/CSS/Components.

## 1. Visual Design Goals
* **Inactive State:** The button should clearly invite action. It needs to say "Boost" text, not just be an icon.
* **Active State ("High Voltage"):** The UI must feel "charged."
    * **Glow:** Neon Green (`#39FF14`) shadows.
    * **Pulse:** The icon should heartbeat.
    * **Speed:** The Odometer text looks "hotter" (brighter green).

## 2. Implementation Specs

### Component A: The Transformative Boost Button
Refactor the center button in the `UnifiedLedger` to change shape based on state.

* **State 1: Inactive (Call to Action)**
    * **Shape:** Small Pill (Capsule).
    * **Content:** `[⚡ Boost]` (Icon + Text).
    * **Style:** `bg-slate-200` (Light Grey), `text-slate-600`, `text-xs font-bold`.
    * **Hover:** `hover:bg-slate-300`.
* **State 2: Active (Timer/Pulse)**
    * **Shape:** Circle (or condensed Pill).
    * **Content:** `[⚡]` (Icon only) OR `[⚡ 59m]` if space permits.
    * **Style:** `bg-neon-green` (`#39FF14`), `text-slate-900`.
    * **Animation:** Infinite "Heartbeat" pulse.
    * **Transition:** Use `LayoutAnimation` (Framer Motion) to smoothly shrink from "Boost" text to just the Icon.

### Component B: The "High Voltage" Container Glow
When active, the "Pending Yield" section of the Unified Console needs to glow.

* **Target:** The White Container holding the Left Yield Pill.
* **CSS Effect (Active):**
    * Apply a `drop-shadow` or `box-shadow` specifically to the **Left Side** of the container.
    * **Tailwind Class:** `shadow-[0_0_20px_rgba(57,255,20,0.6)]` (Neon Green Glow).
    * **Border:** Change the Yield Text color from standard Growth Green (`#00C805`) to **Neon Green** (`#39FF14`) to look "electric."

### Component C: The Activation "Shock"
When the boost initially activates (the moment the modal closes):

* **Trigger:** When `isBoostActive` switches from `false` to `true`.
* **Animation:** A single "Shockwave" ripple effect expanding from the Boost Button.
* **Tech:** Use a simple CSS keyframe animation on a pseudo-element or a Framer Motion one-shot animation.
